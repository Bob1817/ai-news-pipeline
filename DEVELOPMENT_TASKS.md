## 3. 任务详细说明

### 3.1 ENV-001：Python环境配置

**目标**：搭建开发所需的Python环境

**步骤**：
1. 安装Python 3.12+（官网下载安装）
2. 创建项目虚拟环境：`python -m venv venv`
3. 激活虚拟环境
4. 安装pip升级工具

### 3.2 ENV-002：依赖包安装

**目标**：安装所有必要的Python依赖包

**步骤**：
1. 安装PyQt6：`pip install pyqt6`
2. 安装Playwright：`pip install playwright`
3. 安装其他依赖：`pip install python-dotenv pyyaml loguru beautifulsoup4`
4. 验证依赖安装成功

### 3.3 ENV-003：浏览器驱动配置

**目标**：配置Playwright浏览器驱动

**步骤**：
1. 运行：`playwright install chromium`
2. 验证浏览器安装成功
3. 测试浏览器基本功能

### 3.4 UI-001：主窗口框架

**目标**：创建应用主窗口

**步骤**：
1. 创建`src/ui/main_window.py`
2. 继承QMainWindow
3. 创建菜单栏（文件、设置、帮助）
4. 创建标签页容器（采集、文章管理、发布）
5. 创建状态栏（状态显示、日志提示）

**输出**：
- `src/ui/main_window.py`

### 3.5 UI-002：采集配置标签页

**目标**：实现采集配置界面

**步骤**：
1. 创建`src/ui/collect_tab.py`
2. 行业选择下拉框（从配置加载）
3. 主题关键词输入框
4. 采集数量选择（10/20/30）
5. 主题数量选择（3/5/8）
6. 时间范围选择
7. 开始采集按钮
8. 进度条显示
9. 采集结果列表展示

**输出**：
- `src/ui/collect_tab.py`

### 3.6 UI-003：文章管理标签页

**目标**：实现文章管理界面

**步骤**：
1. 创建`src/ui/article_tab.py`
2. 文章列表表格（标题、创建时间、状态）
3. 文章详情展示区域
4. 文章内容编辑器
5. 配图预览
6. 编辑、删除、导出按钮
7. 关键词标签显示

**输出**：
- `src/ui/article_tab.py`

### 3.7 UI-004：发布管理标签页

**目标**：实现发布管理界面

**步骤**：
1. 创建`src/ui/publish_tab.py`
2. 待发布文章列表
3. 发布平台选择（网站、微信公众号）
4. 发布按钮
5. 发布历史记录
6. 发布状态追踪

**输出**：
- `src/ui/publish_tab.py`

### 3.8 UI-005：设置对话框

**目标**：实现系统设置界面

**步骤**：
1. 创建`src/ui/settings_dialog.py`
2. LLM配置区域（提供商、模型、API地址、密钥）
3. 文章参数设置（字数范围、自动配图）
4. 发布设置（审核开关、发布间隔）
5. 保存/取消按钮

**输出**：
- `src/ui/settings_dialog.py`

### 3.9 UI-006：任务进度组件

**目标**：实现统一的任务进度显示组件

**步骤**：
1. 创建`src/ui/task_progress.py`
2. 进度条组件
3. 日志输出区域
4. 任务状态标签
5. 取消按钮

**输出**：
- `src/ui/task_progress.py`

### 3.10 BIZ-001：任务管理器

**目标**：实现异步任务管理

**步骤**：
1. 创建`src/business/task_manager.py`
2. 使用QThreadPool管理线程
3. 定义任务信号（进度、完成、错误）
4. 实现任务提交方法
5. 实现任务取消方法
6. 维护活跃任务字典

**输出**：
- `src/business/task_manager.py`

### 3.11 BIZ-002：流程执行器

**目标**：编排完整的新闻自动化流程

**步骤**：
1. 创建`src/business/pipeline_runner.py`
2. 封装原项目的采集器、分析器、生成器、分发器
3. 实现`run_full_pipeline`方法
4. 实现各步骤的独立方法
5. 添加进度回调

**输出**：
- `src/business/pipeline_runner.py`

### 3.12 BIZ-003：配置管理器

**目标**：管理系统配置

**步骤**：
1. 创建`src/business/config_manager.py`
2. 实现YAML配置文件加载
3. 实现环境变量加载
4. 实现配置保存
5. 提供配置查询接口

**输出**：
- `src/business/config_manager.py`

### 3.13 RES-001：图标资源准备

**目标**：准备应用图标资源

**步骤**：
1. 创建`resources/icons/`目录
2. 创建应用图标`app.ico`
3. 创建功能图标（采集、生成、发布、设置等）
4. 确保图标为SVG或ICO格式

**输出**：
- `resources/icons/app.ico`
- `resources/icons/collect.svg`
- `resources/icons/generate.svg`
- `resources/icons/publish.svg`
- `resources/icons/settings.svg`

### 3.14 RES-002：样式文件开发

**目标**：美化应用界面

**步骤**：
1. 创建`resources/styles/main.qss`
2. 定义全局样式（字体、颜色）
3. 定义按钮、输入框样式
4. 定义表格样式
5. 应用样式到主窗口

**输出**：
- `resources/styles/main.qss`

### 3.15 PKG-001：PyInstaller配置

**目标**：配置打包参数

**步骤**：
1. 创建`pyinstaller.spec`
2. 配置脚本路径
3. 配置数据文件（config、resources、data）
4. 配置隐藏导入
5. 配置图标和窗口模式

**输出**：
- `pyinstaller.spec`

### 3.16 PKG-002：打包测试

**目标**：生成可执行文件

**步骤**：
1. 执行打包命令：`pyinstaller pyinstaller.spec`
2. 验证dist目录生成
3. 测试运行可执行文件
4. 检查依赖是否完整

### 3.17 PKG-003：功能测试

**目标**：验证所有功能正常

**步骤**：
1. 测试采集功能（行业选择、关键词搜索）
2. 测试分析功能（主题提炼）
3. 测试文章生成（内容生成、配图生成）
4. 测试发布功能（网站、微信公众号）
5. 测试设置功能（配置保存加载）

### 3.18 PKG-004：Bug修复与优化

**目标**：修复问题，优化性能

**步骤**：
1. 修复测试中发现的Bug
2. 优化UI响应速度
3. 优化任务执行效率
4. 优化内存使用
